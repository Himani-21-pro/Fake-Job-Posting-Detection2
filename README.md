# 💼 Fake Job Posting Detection System

This project is a Machine Learning-based web application that detects whether a job posting is **real or fake** using Natural Language Processing (NLP) techniques.

---

## 🚀 Features
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Classification using Logistic Regression and Naive Bayes
- Interactive web application using Streamlit
- Real-time prediction of job postings

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## 📊 Model Performance
- Compared multiple models (Logistic Regression, Naive Bayes)
- Selected best model based on accuracy and performance

---

1. Install dependencies:

2. Run the app:
